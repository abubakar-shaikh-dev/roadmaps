- Modifiers (signed, unsigned, short, long)

- Declaring and initializing variables
- Using constants and literals

- Arithmetic, relational, and logical operators

## Control Statements 🔀

## Arrays and Strings 🎵

- Manipulating strings and string functions

## Functions 📋

## Pointers 🔍

- Pointer arithmetic
